# CarMart

**Short Description:** A responsive web showcase for car listings with a trade‑in contact form, featuring premium UI design, dynamic car tiles, and mobile‑first layout.

## Overview
CarMart is a lightweight HTML/CSS website that displays car listings in an elegant, responsive grid. It includes:
- Horizontal scrolling car tiles on mobile for a smooth "side‑sweep" experience.
- Consistent, premium styling with modern CSS variables, glass‑morphism, and micro‑animations.
- A trade‑in form allowing users to submit vehicle details.

## Features
- Responsive design that adapts to desktop, tablet, and mobile screens.
- Horizontal carousel‑style layout for car cards on small devices.
- Inline "Order Now" buttons aligned across all cards.
- Accessible navigation with a hamburger menu.
- Easy customization via the shared `style.css`.

## Getting Started
1. Clone the repository:
   ```bash
   git clone <repo-url>
   ```
2. Open `CAR.html` (or any other page) in a browser.
3. No build step is required – the site runs purely with static HTML/CSS.
